# Syntax Variable

In general, you can use variables in javascript like this

```js
const a = "Kulo"
var b = "Sampean"
let c = 5
let d;
```

But you can also use it here like this

```js
konstan data = "Kulo"
konstan dataOne iku "Kulo"
variable dataTelu = "Sampean"
variable dataPapat iku "Sampean"
benkedata dataLima = 5
benkedata dataEnem;
```
